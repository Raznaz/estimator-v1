## Summary

Добавляет эндпоинт сводки по транзакциям в бэкенд. Контроллер обращается к данным через `QueryBus` (CQRS), агрегируя суммы по категориям за период, чтобы фронтенд мог отрисовать дашборд одним запросом вместо нескольких.

## Changes

- Добавлен `GET /transactions/summary` с query-параметрами `from`/`to`.
- Добавлен `GetTransactionsSummaryQuery` + handler с агрегацией через Prisma `groupBy`.
- Покрыто unit-тестами: пустой период, группировка по категориям, фильтр по датам.
